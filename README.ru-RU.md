# The PostgreSQL Framework for the backend development.

Программная платформа (framework) поверх PostgreSQL для разработки серверной части автоматизированных информационных систем.

Описание
-

Подробное описание доступно в [Wiki](https://github.com/apostoldevel/db-platform/wiki).

Реализованные проекты
-

Реализованные проекты на данной платформе:

* [Ship Safety ERP](https://ship-safety.ru) - Автоматизированная ERP-система управления безопасностью для судоходных компаний;
* [CopyFrog](https://copyfrog.ai) - Платформа на базе искусственного интеллекта для создания уникальных изображений, рекламных текстов, видеокреативов и маркетинговых описаний для продуктов и услуг;
* [Talking to AI](https://t.me/TalkingToAIBot) - Чат-бот в Telegram для общения с искусственным интеллектом;
* [OCPP CSS](http://ocpp-css.ru) - Облачная SaaS-платформа для управления зарядными станциями для электромобилей;
* [PlugMe](https://plugme.ru) - Система управления зарядными станциями/Charging Station Management System (CSMS);
* [DEBT-Master](https://debt-master.ru) - Система для автоматизации работ с дебиторской задолженностью потребителей – должников за коммунальные услуги;
* [BitDeals](https://testnet.bitdeals.org/info/about) - Система арбитража для совершения сделок с использованием криптовалюты BTC.
